Ansible Vips Deployment
#######################

The ansible role (not currently) provided deploys vips and the associated python bindings to the server

Vips is only an optional requirement of exif2timestream-v2 and it will fall back to Pillow

Requirements
------------

.. highlight:: console

You need `Ansible <https://www.ansible.com/>`_ version 2.3 installed locally

.. code-block:: bash

    # pip install git+https://github.com/ansible/ansible
